# 安全检查清单

> 审查时按场景扫描变更代码是否命中。命中违规标 P0（安全漏洞）或 P1（安全隐患）。仅检查与变更相关的场景，不相关场景标 N/A。

---

## S1 SQL 注入

| ID | 检查项 | 等级 |
|---|--------|------|
| S1.1 | SQL 参数使用预编译 `#{}`，禁止 `${}` 拼接 | P0 |
| S1.2 | order by / group by / 动态表名字段名使用白名单 | P0 |
| S1.3 | like 查询使用 CONCAT 参数化，in 查询使用 foreach | P1 |

## S2 XSS

| ID | 检查项 | 等级 |
|---|--------|------|
| S2.1 | HTML/JS/URL/JSON 输出按场景编码或转义 | P0 |
| S2.2 | 富文本内容过滤（白名单标签） | P1 |
| S2.3 | 模板引擎使用安全宏/自动转义 | P1 |

## S3 SSRF

| ID | 检查项 | 等级 |
|---|--------|------|
| S3.1 | 外部 URL 请求：域名白名单 + 内网 IP 拦截 | P0 |
| S3.2 | 302 跳转后目标地址重新校验 | P1 |
| S3.3 | 超时已设置 | P1 |

## S4 命令执行

| ID | 检查项 | 等级 |
|---|--------|------|
| S4.1 | 禁止外部参数拼接系统命令 | P0 |
| S4.2 | 文件/图片操作使用 Java API，不调用外部命令 | P1 |

## S5 XXE

| ID | 检查项 | 等级 |
|---|--------|------|
| S5.1 | XML 解析器禁用外部实体和 DTD | P0 |
| S5.2 | XPath 用户输入已过滤 | P1 |

## S6 反序列化

| ID | 检查项 | 等级 |
|---|--------|------|
| S6.1 | 反序列化数据来源可信，已配置类白名单 | P0 |
| S6.2 | JSON 反序列化禁用多态（`@JsonTypeInfo` 白名单） | P0 |
| S6.3 | 敏感字段 `transient` 修饰 | P1 |

## S7 文件上传/下载

| ID | 检查项 | 等级 |
|---|--------|------|
| S7.1 | 文件后缀白名单 + 大小限制 | P1 |
| S7.2 | 路径过滤 `../`，防止目录穿越 | P0 |
| S7.3 | 文件重命名存储，权限校验 | P1 |

## S8 访问控制

| ID | 检查项 | 等级 |
|---|--------|------|
| S8.1 | 接口已接入鉴权，资源操作校验归属权 | P0 |
| S8.2 | 禁止 GET 执行增删改 | P1 |
| S8.3 | 数据 ID 不可预测（UUID 或加密） | P1 |
| S8.4 | Cookie 设置 HttpOnly + Secure | P1 |

## S9 数据安全

| ID | 检查项 | 等级 |
|---|--------|------|
| S9.1 | 密钥/凭证不硬编码，从配置中心获取 | P0 |
| S9.2 | 日志不记录敏感信息（密码、token、身份证等） | P0 |
| S9.3 | 传输加密（HTTPS）、存储加密、使用安全算法 | P1 |
| S9.4 | 随机数使用 `SecureRandom` | P1 |

## S10 CSRF / CORS / 跳转

| ID | 检查项 | 等级 |
|---|--------|------|
| S10.1 | 增删改操作有 CSRF Token 防护 | P1 |
| S10.2 | CORS `Access-Control-Allow-Origin` 白名单，禁止 `*` | P1 |
| S10.3 | URL 跳转目标白名单校验 | P0 |
