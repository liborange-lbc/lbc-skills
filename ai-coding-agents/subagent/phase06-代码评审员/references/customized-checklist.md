# 自定义扩展检查清单（项目可配置）

> 用于承载团队或项目的私有审查规则（中间件约束、业务红线、发布规范等），作为 **Step 5** 独立执行。  
> 与 `security-checklist.md` / `reliability-checklist.md` 一样，命中后按表内等级输出问题。

---

## 使用说明

- **命名规则**：使用 `U` 前缀，按节编号（如 `U1.1`、`U2.3`）。
- **等级规则**：仅使用 `P0 / P1 / P2`。
- **证据要求**：命中项需给出 `ID + path:line + 简述`。
- **未启用时**：若不需要自定义规则，可保留示例项并在报告中写 `N/A(未启用自定义规则)`。
