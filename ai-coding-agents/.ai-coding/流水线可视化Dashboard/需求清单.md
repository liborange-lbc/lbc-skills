# 需求确认清单 — 流水线可视化Dashboard

## 业务背景

ai-coding-agents 技能定义了 8 Phase 多 Agent 编码流水线（P1 需求澄清 → P2 知识采集 → P3 系分编写 → P4 系分评审 → P5 编码实现 → P6 代码CR → P7 测试验证 → P8 交付报告）。目前流水线运行状态不可见，开发者需要一个本地浏览器可视化 Dashboard，既能展示流水线设计理念（融入流程展示），又能在运行时读取 log 目录的实际数据展示实时状态。

技术方案：纯原生 HTML/CSS/JS（零依赖），Python HTTP Server 扫描 log 目录返回 JSON，前端每 5 秒轮询刷新。

## 非功能需求

| ID | 描述 | 指标 |
|----|------|------|
| NFR01 | 零外部依赖 | 前端不引入任何 npm 包、CDN 资源或第三方 JS 库；后端仅使用 Python 标准库 |
| NFR02 | 本地可运行 | 一条命令启动（如 `python server.py`），浏览器访问 localhost 即可使用 |
| NFR03 | 轮询性能 | 每 5 秒一次轮询请求，响应时间 < 500ms（本地文件扫描） |
| NFR04 | 浏览器兼容性 | 支持 Chrome/Safari/Firefox 最新版 |

## 约束条件

- 前端：纯原生 HTML/CSS/JavaScript，禁止引入任何 npm 包或 CDN
- 后端：Python 标准库（http.server + json + os/pathlib），禁止引入第三方包
- 数据源：`.ai-coding/{需求名称}/log/` 目录下的文件
- 运行环境：开发者本地浏览器（macOS）
- Markdown 渲染：纯 JS 手写解析器，支持标题、列表、代码块、表格、链接即可
